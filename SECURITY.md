# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Arrow Thing, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **security@arrow-thing.com** or report via [GitHub's private vulnerability reporting](https://github.com/vicplusplus/arrow-thing/security/advisories/new).

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

## Response

- We will acknowledge receipt within 48 hours.
- We will provide an initial assessment within 7 days.
- We will work with you to understand and resolve the issue before any public disclosure.

## Scope

The following are in scope:
- The game server API at `api.arrow-thing.com`
- The web client at `arrow-thing.com`
- This GitHub repository and its CI/CD pipelines

The following are out of scope:
- Cloudflare infrastructure
- Third-party services (Resend, Discord)
- Denial-of-service attacks

## Supported Versions

Only the latest deployed version is supported. There are no LTS or backport branches.
